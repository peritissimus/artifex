About TOON Format

        

          Token-Oriented Object Notation (TOON) is a compact, human-readable
          encoding of the JSON data model for LLM prompts. It provides a lossless serialization
          using fewer tokens than JSON by combining YAML's indentation-based structure with
          CSV-style tabular arrays.
