Brihaspati Mobile App

Company: Brihaspati Role: Founding Engineer Period: May 2024 - November 2024

Overview

As Founding Engineer, I led the development of Brihaspati's AI-powered mobile application from the ground up. Built with clean architecture principles and delivered within 2 months, the app provides a unique user experience backed by scalable infrastructure handling 100K+ daily requests.

Key Achievements

  • Led development of mobile application using clean architecture
  • Collaborated with designers and product consultants to create unique UX within 2 months
  • Architected scalable backend infrastructure handling 100K+ daily requests
  • Maintained 99.9% uptime through robust system design
  • Implemented dynamic prompt engineering and context-aware memory retrieval
  • Achieved 20% improvement in D7 retention through personalized content generation

Technical Architecture

The app follows clean architecture principles with clear separation of concerns. The mobile frontend is built with Flutter and Bloc for state management, while the backend leverages Django, PostgreSQL, and AWS for scalability and reliability.

AI Features

Implemented sophisticated AI features including dynamic prompt engineering and context-aware memory retrieval systems. These features enable personalized content generation that adapts to user behavior and preferences, significantly improving user engagement and retention.

Technical Stack

Flutter Dart Bloc Django PostgreSQL AWS Firebase AI/ML
